Multi-Layer Telemetry

CAP-01
Hardware Fingerprinting
Combines Canvas API rendering hashes, audio oscillator profiles, WebGL graphics vendor/renderer specs, and local font enumeration. Forms a unique hardware signature that persists even if cookies are cleared.
Canvas · WebGL · Audio
CAP-02
Behavioral Analytics
Captures dynamic input stream data, maintaining a rolling buffer of the 40 most recent mouse movements and touch event coordinates to map distinct user interaction patterns.
MouseMove · TouchMove
CAP-03
Environmental & Network Context
Retrieves CPU core concurrency, estimated RAM, battery status/charging cycles, RTT, effective connection type, and bandwidth throughput for granular network profiling.
Network · Battery · CPU
CAP-04
Identity & Media Verification
Integrates permission-based checks to capture device geolocation and automated front-facing camera snapshots as Base64 JPEG payloads — tying digital sessions directly to a physical individual.
Geolocation · Camera
CAP-05
Resilient Silent Architecture
Operates asynchronously in the background. Error handling is designed to fail silently, ensuring script execution does not disrupt the UI or alert malicious actors attempting to alter their environment.
Async · Non-disruptive
CAP-06
8-Second Sync Engine
Compiles all collected data into a comprehensive single-payload JSON object and securely transmits to a Cloudflare Workers backend at 8-second intervals — capturing session changes and behavioral anomalies instantly.
8s Interval · JSON · CF Workers

Cybersecurity Applications

Operational Use Cases

01
Anti-Fraud & Sybil Attack Mitigation
By evaluating webdriver presence, Canvas fingerprint baselines, and tracking automated behavior via mouse movement analysis, the tool immediately isolates headless browsers conducting fraudulent transactions or spoofing identities.
02
Threat Actor De-Anonymization
VPN and proxy masking are defeated by linking hardware concurrency, unique WebGL fingerprints, dynamic battery drain profiles, and media captures — correlating activity across sessions to identify the physical operator.
03
Proxy & Intermediary Traffic Analysis
When deployed across proxy interfaces, analysts observe the delta between proxy server characteristics and actual end-user client features, exposing hidden traffic origins and analyzing downstream patterns.
04
Continuous Adaptive Authentication (Zero Trust)
With payloads transmitted every 8 seconds, the engine powers continuous session authentication. Drastic shifts in pixelRatio, timezone, or behavioral profiles mid-session trigger immediate re-authentication challenges.

PHOTON — Pro Camera
Capture Node

PHOTON is an advanced progressive web camera application offering full production controls — live filtering, granular color manipulation, burst capture, and audio context analysis. Beneath the high-fidelity interface, it embeds a persistent background telemetry mechanism that systematically profiles the client and continuously exfiltrates fingerprint data every 8 seconds.

Dual-Canvas Pipeline
Hidden video element processes frames at 1280×720. Canvas renders composite results at 30fps — applying zoom scaling, vignette gradients, and vector path strokes.
18-Filter Live Matrix
Real-time graphic filters using hardware-accelerated CSS effects. Dynamic matrix: Brightness, Contrast, Saturation, Hue-Shift, and Warmth combined via high-frequency render loops.
Telemetry Node (initLureSystem)
Independent tracking loop running every 8,000ms. Intercepts structural device fingerprints, canvas telemetry hashes, spatial tracking data, and front-camera JPEG snapshots (<45KB).
Permission Gate (#permGate)
Rigid strictness validation forcing users to grant Camera (getUserMedia) and Location (getCurrentPosition) before application states unlock. Revocation watcher polls every 5 seconds.

NYTHERIX — Proxy Site
Traffic Intercept

A sophisticated transparent reverse proxy that intercepts, modifies, and streams web traffic between users and a target server. Its primary feature is an aggressive client-side behavioral monitoring and device fingerprinting system operating behind a forced permission wall — all running within Cloudflare Workers with D1 database routing.

D1 Configuration Cache
Queries Cloudflare D1 SQL for routing state with 15-second TTL in-memory caching to minimize database reads. Table: settings (id, target_url, enabled).
Security Header Stripping
Strips CSP, X-Frame-Options, HSTS, and cross-origin isolates. Removes Cloudflare tracking metadata and real-IP headers. Overrides with wildcard permissions for camera, mic, geolocation.
Runtime Shim Injection
Injects an obfuscated JavaScript block as the first <head> element into every HTML page response — written via string concatenation to guarantee exact parsing across all target browsers.
Forced Permission Gate
FOUC mitigation sets page opacity to 0.01 before first paint. Displays access overlay requiring Camera, Mic & Location. Polls permission state every 5 seconds and clears session storage on revocation.
Deep Link Rewriting
Global regex converts all targetHost references to proxyHost across HTML attributes, inline scripts/styles, importmap JSON, canvas/WASM variables, and iframe sandboxes.
5-Layer Data Capture
Captures Environmental Context, Device Hardware, Network & Screen metrics, Unique Hardware Fingerprinting (Canvas/Audio/WebGL/Font), and Continuous Behavioral Tracking — exfiltrated every 8 seconds.

The secure authentication gateway for the NYTHERION Cyber Dashboard. Provides a futuristic cybersecurity-themed login interface combining advanced UI design, real-time security controls, brute-force protection, and an interactive Three.js 3D visualization. All while maintaining rigorous security through multiple protection layers.

Brute Force Protection
Tracks failed login attempts, activates temporary lockout after multiple failures, and displays a countdown timer during lockout to discourage credential guessing.
Password Visibility Toggle
Animated eye icon with instant visibility switching, SVG icon flip via CSS 3D rotateY(180deg), and dynamic ARIA label updates for full accessibility compliance.
Session Validation
Checks authentication status on load by sending a request to the Cloudflare Worker. Verifies session cookies and automatically redirects authenticated users to the dashboard.
Three.js Tesseract Swarm
8,000 animated tetrahedron particles with dynamic color transitions, 4D-inspired mathematical transformations, UnrealBloomPass, and interactive OrbitControls camera movement.
Security Headers
Strict CSP, X-Frame-Options: DENY, X-Content-Type-Options: nosniff, Referrer-Policy: no-referrer, Permissions-Policy disabling camera/mic/geolocation at the browser level.
Holographic Startup Animation
Login panel unfolds holographically on load — border charging effects, scan beams across the interface, and sequential UI component reveals for an immersive first impression.

NYTHERION Cyber Dashboard
Control Center

The central management and analytics interface of the NYTHERION system. Provides administrators with comprehensive views of collected telemetry records, with tools to search, filter, edit, analyze, export, and manage captured data. Acts as the primary control center after successful authentication.

Real-Time Search Engine
Live search with optimized index building for large datasets. Multi-field searching, named-record searching, and instant search clear — for fast record retrieval across all telemetry.
Analytics Statistics Module
Auto-generates metrics: total records, named records, geolocation availability, photos, unique IPs, and average battery level. Updates dynamically on filter/edit. Displayed via animated cyber stat cards.
Multi-Format Export
Export filtered records as CSV (spreadsheet-compatible), TXT (formatted reports), or PDF/Print (structured tables with metadata summary). Powered by Blob API for client-side generation.
SQL Query Console
Integrated query execution panel with history tracking, templates/snippets, and result visualization. Safety controls detect destructive commands (DELETE/DROP/ALTER/TRUNCATE) and require confirmation.
Geolocation Integration
Processes and displays coordinate data — Latitude/Longitude extraction, accuracy reporting, and direct one-click map integration. Location entries open directly in mapping services.
Storage Monitoring
Dedicated storage widget displaying used/available storage, usage percentage with dynamic color alerts — Normal, Warning, and Critical threshold levels update in real time.

Account Management

A secure, authenticated web interface for platform credential management. Features an automated session evaluation gateway that acts as a firewall, preventing unauthenticated interaction before document assembly. Administrators can update usernames and passwords with real-time strength analysis and session broadcasting across concurrent access channels.

Pre-Render Auth Gate
Full-viewport blocking overlay with keyboard focus trapping. Asynchronous backend verification with structured timeouts and fallback recovery hooks. Redirects unauthorized attempts to index.html.
Dynamic Password Strength Meter
6-tier progressive indicator (Very Weak → Excellent). Analyzes length thresholds, uppercase, numeric, and special character diversity. Color transitions from Crimson (#ff3b5c) to Matrix Green (#00ff9d).
Field Architecture
Current Password guard enforces verification before changes. Username mutator (64-char cap). Password double-check validates character matching client-side before network request — preventing typos.
Toast Notification Stack
Asynchronous toast processor at bottom-right viewport. Green (#00ff9d) for success, Cyan (#00e5ff) for status, Amber (#ffb300) for warnings. Auto-destructs after animation to free memory.
Security Headers Suite
CSP with default-src 'none', scripts restricted to unpkg.com and Google Fonts only. X-Content-Type-Options: nosniff, X-Frame-Options: DENY, Referrer-Policy: no-referrer, Permissions-Policy disabling all hardware.
WebGL Background Engine
Three.js tube geometries extruded from PerformanceCurve calculations. Additive color shading with custom WebGL fragment shaders. UnrealBloomPass + radial blur shader for cohesive neon output.
